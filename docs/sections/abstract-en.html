The Academic Research Publication Ontology (ARPO) is an OWL 2 DL domain ontology engineered to formally represent the complete lifecycle and ecosystem of scholarly publishing. ARPO provides a unified semantic framework modeling key domain aspects including publication types (journal articles, conference papers, preprints), entities (authors, researchers, institutions, research groups), and relationships (citations, funding grants, and open access status). Developed using a modular design approach, the ontology addresses modern bibliometric and semantic web needs, enabling sophisticated semantic query execution, data interoperability across disparate bibliographic data stores, and automated knowledge graph population pipelines.
